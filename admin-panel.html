🚀 Firebase Setup Required

  1. Go to firebase.google.com and create a free account
  2. Create new project called "Samad Bags"
  3. Enable Firestore Database (start in test mode)
  4. Enable Authentication → Sign-in method → Email/Password
  5. Go to Project Settings → Your apps → Web app → Register
  6. Copy the config values and paste in firebase-config.js
  7. Create an admin user in Authentication → Users → Add User
  8. Refresh this page and login with your admin credentials
